# Movie Ticket Booking System (The Movies)

## Project Outline
A web application allowing users to book movie tickets, select dates, and choose seats dynamically without reloading the page.

## Development Environment
- Python 3.9+
- Package Manager: uv
- Framework: Django

## Tools Used
- Formatting: Black
- Linting: Pylint (with pylint-django)
- Testing: pytest, pytest-django, coverage
